# thunderdome
## setup

#### 1.clone repository
```
$ git clone git@github.com:kjugk/thunderdome.git
```

#### 2. install gems
```
$ cd thunderdome
$ bundle install --path vendor/bundle
```

#### 3. setup database
```
$ rake db:setup
```

#### 4. start server
```
$ rails s
```
